# Prettier configuration file reference

## Config

_Object containing the following properties:_

| Property                     | Description                                                                                                                                              | Type                                                                                                                                                                                                                                                                                                                                                                       | Default       |
| :--------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------ |
| `arrowParens`                | Include parentheses around a sole arrow function parameter.                                                                                              | `'always' \| 'avoid'` (_default:_ `'always'`)                                                                                                                                                                                                                                                                                                                              | `'always'`    |
| `bracketSameLine`            | Put > of opening tags on the last line instead of on a new line.                                                                                         | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `bracketSpacing`             | Print spaces between brackets.                                                                                                                           | `boolean` (_default:_ `true`)                                                                                                                                                                                                                                                                                                                                              | `true`        |
| `cursorOffset`               | Print (to stderr) where a cursor at the given position would move to after formatting.                                                                   | `number` (_int_) (_default:_ `-1`)                                                                                                                                                                                                                                                                                                                                         | `-1`          |
| `embeddedLanguageFormatting` | Control how Prettier formats quoted code embedded in the file.                                                                                           | `'auto' \| 'off'` (_default:_ `'auto'`)                                                                                                                                                                                                                                                                                                                                    | `'auto'`      |
| `endOfLine`                  | Which end of line characters to apply.                                                                                                                   | `'lf' \| 'crlf' \| 'cr' \| 'auto'` (_default:_ `'lf'`)                                                                                                                                                                                                                                                                                                                     | `'lf'`        |
| `experimentalTernaries`      | Use curious ternaries, with the question mark after the condition.                                                                                       | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `filepath`                   | Specify the input filepath. This will be used to do parser inference.                                                                                    | `string`                                                                                                                                                                                                                                                                                                                                                                   |               |
| `htmlWhitespaceSensitivity`  | How to handle whitespaces in HTML.                                                                                                                       | `'css' \| 'strict' \| 'ignore'` (_default:_ `'css'`)                                                                                                                                                                                                                                                                                                                       | `'css'`       |
| `insertPragma`               | Insert @format pragma into file's first docblock comment.                                                                                                | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `jsxSingleQuote`             | Use single quotes in JSX.                                                                                                                                | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `parser`                     | Which parser to use.                                                                                                                                     | `'flow' \| 'babel' \| 'babel-flow' \| 'babel-ts' \| 'typescript' \| 'acorn' \| 'espree' \| 'meriyah' \| 'css' \| 'less' \| 'scss' \| 'json' \| 'json5' \| 'jsonc' \| 'json-stringify' \| 'graphql' \| 'markdown' \| 'mdx' \| 'vue' \| 'yaml' \| 'glimmer' \| 'html' \| 'angular' \| 'lwc' \| string`                                                                       |               |
| `plugins`                    | Add a plugin. Multiple plugins can be passed as separate `--plugin`s.                                                                                    | `Array<string>` (_default:_ `[]`)                                                                                                                                                                                                                                                                                                                                          | `[]`          |
| `printWidth`                 | The line length where Prettier will try wrap.                                                                                                            | `number` (_int_) (_default:_ `80`)                                                                                                                                                                                                                                                                                                                                         | `80`          |
| `proseWrap`                  | How to wrap prose.                                                                                                                                       | `'always' \| 'never' \| 'preserve'` (_default:_ `'preserve'`)                                                                                                                                                                                                                                                                                                              | `'preserve'`  |
| `quoteProps`                 | Change when properties in objects are quoted.                                                                                                            | `'as-needed' \| 'consistent' \| 'preserve'` (_default:_ `'as-needed'`)                                                                                                                                                                                                                                                                                                     | `'as-needed'` |
| `rangeEnd`                   | Format code ending at a given character offset (exclusive).<br />The range will extend forwards to the end of the selected statement.                    | `number` (_int_) (_nullable_) (_default:_ `null`)                                                                                                                                                                                                                                                                                                                          | `null`        |
| `rangeStart`                 | Format code starting at a given character offset.<br />The range will extend backwards to the start of the first line containing the selected statement. | `number` (_int_) (_default:_ `0`)                                                                                                                                                                                                                                                                                                                                          | `0`           |
| `requirePragma`              | Require either '@prettier' or '@format' to be present in the file's first docblock comment<br />in order for it to be formatted.                         | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `semi`                       | Print semicolons.                                                                                                                                        | `boolean` (_default:_ `true`)                                                                                                                                                                                                                                                                                                                                              | `true`        |
| `singleAttributePerLine`     | Enforce single attribute per line in HTML, Vue and JSX.                                                                                                  | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `singleQuote`                | Use single quotes instead of double quotes.                                                                                                              | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `tabWidth`                   | Number of spaces per indentation level.                                                                                                                  | `number` (_int_) (_default:_ `2`)                                                                                                                                                                                                                                                                                                                                          | `2`           |
| `trailingComma`              | Print trailing commas wherever possible when multi-line.                                                                                                 | `'all' \| 'es5' \| 'none'` (_default:_ `'all'`)                                                                                                                                                                                                                                                                                                                            | `'all'`       |
| `useTabs`                    | Indent with tabs instead of spaces.                                                                                                                      | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| `vueIndentScriptAndStyle`    | Indent script and style tags in Vue files.                                                                                                               | `boolean` (_default:_ `false`)                                                                                                                                                                                                                                                                                                                                             | `false`       |
| **`overrides`** (\*)         | Provide a list of patterns to override prettier configuration.                                                                                           | _Array of objects:_<ul><li><b><code>files</code></b> (\*): <code>string \| Array<string></code> - Include these files in this override.</li><li><code>excludeFiles</code>: <code>string \| Array<string></code> - Exclude these files from this override.</li><li><code>options</code>: <a href="#options">Options</a> - The options to apply for this override.</li></ul> |               |

_(\*) Required._
